# Автотесты для Stellar Burgers

Учебный проект для тестирования сайта https://stellarburgers.education-services.ru/ с использованием Selenium и pytest.

## Функционал:
- Регистрация
- Вход
- Переходы между страницами
- Конструктор бургеров

## Технологии:
- Python
- Selenium WebDriver
- pytest
